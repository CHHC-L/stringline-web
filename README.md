# Stringline (web build)

Compiled, static build of **Stringline** — a deterministic logistics-hub optimization game.

- Play: https://chhc-l.github.io/stringline-web/
- This repository hosts **compiled artifacts only**. The Stringline source is proprietary and
  lives in a separate private repository.

These files are published automatically from the source repo on every push to `main`. Do not edit
anything here by hand — the whole tree is overwritten on each deploy.
